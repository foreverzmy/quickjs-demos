# quickjs demos

## Init Repo

```sh
git submodule update --init
```

## Build quickjs

```sh
cd quickjs && make
```

## Demo01

Use Quickjs Run JS Code and Call JS Function In C.

```sh
cd demo1
make clean && make && ./main
```

Use Quickjs Run JS Code and Call C Function In JS.
