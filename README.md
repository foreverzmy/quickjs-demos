# quickjs demos

## Init Repo

```sh
git submodule update --init
```

## Build quickjs

```sh
cd quickjs && make
```

## Demo01

Use Quickjs Run JS Code and Call JS Function In C.

```sh
cd demo01
make clean && make && ./main
```

## Demo02

Use Quickjs Run JS Code and Call C Function In JS.

```sh
cd demo02
make clean && make && ./main
```

## Demo03

Use Quickjs Run JS Code with Console API.

```sh
cd demo03
make clean && make && ./main
```
